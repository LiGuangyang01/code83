### 分支介绍

- `main`分支为君哥讲解实现后的分支（重构后的代码）

- `init`分支为阿里云`codeup`大赛原代码分支（重构前的比赛代码）

- 第一关：`weather-machine-baseline`，重构智能农业系统

- 第三关：`poker`，重构德州扑克评判系统

- 总决赛：`store`，重构
